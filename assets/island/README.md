# Island scene assets

Drop transparent PNGs / JPGs here (ideally @2x). The scene loads these paths first and falls back to procedural placeholders if a file is missing.

| File | Purpose |
|------|---------|
| `sky.jpg` | Sky background |
| `clouds-far.png` | Distant cloud layer |
| `clouds-mid.png` | Mid cloud layer |
| `clouds-near.png` | Near cloud layer |
| `ocean-base.jpg` | Ocean color / albedo |
| `foam.png` | Shore foam |
| `normal.png` | Water normal map |
| `water-mask.png` | White = water, black = land (optional) |
| `islands.png` | Land / beaches / cliffs above water |
| `waterfall.png` | Soft-alpha waterfall strip |
| `ship-hull.png` | Ship body (no sails) |
| `sail-main.png` | Main sail |
| `sail-fore.png` | Forward sail |
| `palm-1.png` … `palm-3.png` | Palm trees |
| `treasure-chest.png` | Treasure / cave mouth |
| `seagull.png` | Seagull sprite (or sheet) |
